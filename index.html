PROJECT PROPOSAL

Project Description

There are gender biased phrases for the being used consciously or unconsciously. Considering that the purpose of this project is to neutralize gender discrimination. The program will prove corrections or recommendations if there are some problems detected. The analyses will be in both vocabulary and context scopes.

It will be Web based application where the texts can be given for the analyses or upload file for the same purpose. (NLP) It will be applicable as chrome extension (extendible for other browsers). Mail can be analyses and improved by given recommendations.
